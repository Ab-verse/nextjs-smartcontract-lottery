This is a [Next.js](https://nextjs.org/) Web3 Lottery project 

## Getting Started

1. Install packeges and dependencies
```bash
npm install
#or
yarn
``` 

2. Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
